Watch Only extension uses mempool.space
For use with "account Extended Public Key" https://iancoleman.io/bip39/
Created by, Ben Arc (using, Embit)

GET /watchonly/api/v1/wallet
Headers
{"X-Api-Key": <invoice_key>}
Body (application/json)
Returns 200 OK (application/json)
[<wallets_object>, ...]
Curl example
curl -X GET {{ request.base_url }}watchonly/api/v1/wallet -H "X-Api-Key: {{ user.wallets[0].inkey }}" GET /watchonly/api/v1/wallet/<wallet_id>
Headers
{"X-Api-Key": <invoice_key>}
Body (application/json)
Returns 201 CREATED (application/json)

{"X-Api-Key": <admin_key>}
